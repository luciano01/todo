# To Do
Simple to do list app to manage tasks.

## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/todo.git
$ cd todo
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd todo
$ code .
```
